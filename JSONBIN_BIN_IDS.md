# JSONBin.io Bin IDs

Your actual bin IDs from JSONBin.io:

## Set these in Vercel Environment Variables:

```
JSONBIN_REFERENCES_BIN_ID=6936a950d0ea881f401a6f82
JSONBIN_BLOGS_BIN_ID=69369e9143b1c97be9df4def
JSONBIN_TESTIMONIALS_BIN_ID=69369e92d0ea881f401a5c39
```

## How to add in Vercel:

1. Go to your Vercel project dashboard
2. Settings → Environment Variables
3. Add each variable above
4. Redeploy

This will make reads instant (no need to search by name).
